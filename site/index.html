Coding agents, under real user correction.

TogetherBench is a benchmark built from real multi-turn developer sessions — measuring how well coding agents recover, course-correct, and finish the job when a human keeps nudging them across many turns.

Real sessions on public repositories (20+ stars) from DataClaw, pi, and Hyperswitch, reconstructed as Harbor tasks with Docker environments and deterministic verifiers.

Overview

Most benchmarks test one prompt. Real work is a conversation.

Single-turn benchmarks reward an agent for nailing a perfect first attempt. But day-to-day, engineers steer agents — they correct, clarify, and review across a long session. TogetherBench captures that loop, sourced verbatim from real sessions.

01

Derived from real sessions

Every task is reconstructed from an actual developer session on a public repo — the instruction is the verbatim first user message, not a synthetic prompt.

02

Multi-turn by construction

A user simulator replays the original correction loop — follow-up questions, course pivots, and reviews — so the headline metric is the gain from turn 0 to final.

03

Deterministic verifiers

Each task ships fail-to-pass behavioral tests plus weighted completeness goals — partial credit, implementation-agnostic, no LLM in the scoring path.
